WP Dashboard Chat is a Twitter inspired widget, which displays on the admin dashboard. It allows for quick communication between users and admins.

== Description ==

WP Dashboard Chat allows admins to communicate via a widget which displays on the Admin Dashboard. Itâ€™s perfect for quickly getting messages out to all your sites writers or having a formal meeting. Features include, @ mentions and replies, live updating, Gravatar support, auto-linking and HTML formatting. Many more coming soon!

== Installation ==

1. Upload the 'wp-dashboard-chat' folder to the `/wp-content/plugins/` directory or install via the wordpress plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit the Dashboard home page to start using the plugin

== Frequently Asked Questions == 

= Is the chat live? =

Yes! Well, pretty close to it. As of now, it updates every few seconds.

= Can I set the minimum user level who can see the chat? =

Not yet, but in a version coming very soon, you will be able to.
